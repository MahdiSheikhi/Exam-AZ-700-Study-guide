Exam Overview

The AZ-700 exam tests your expertise in designing, implementing, and managing Azure networking solutions. This includes core infrastructure, hybrid connectivity, application delivery, private access, and network security.

Practice Questions

Test your knowledge with exam-style questions

Core Networking

What is the maximum number of subnets you can create in a single VNet?

A) 1000
B) 2000
C) 3000
D) 5000
Correct! Azure allows up to 3000 subnets per virtual network, providing ample capacity for complex network architectures.
VPN Gateway

Which VPN Gateway SKU supports active-active mode for high availability?

A) Basic
B) VpnGw1
C) VpnGw2
D) All SKUs
Correct! VpnGw2 and higher SKUs support active-active mode, providing redundancy and improved performance for VPN connections.
Application Gateway

At which OSI layer does Azure Application Gateway operate?

A) Layer 3 (Network)
B) Layer 4 (Transport)
C) Layer 7 (Application)
D) Layer 2 (Data Link)
Correct! Application Gateway is a Layer 7 load balancer that provides advanced application delivery controller functionality including SSL termination, path-based routing, and web application firewall capabilities.
